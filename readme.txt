== Description ==

Add a free customized mortgage widget to your site. Mortgage rates update automatically daily. You can change the color of the background, fields, and calculate button to match your site's theme.

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance -> Widgets and add the widget
